## :rocket: Sobre a Aplicação

O GoFinances Web é uma aplicação criada com React JS para gestão de transações financeiras.

## Instalação

Instalação das dependências do projeto.

```sh
# Com NPM
npm install

# Com Yarn
yarn
```

## Executando a Aplicação

```sh
# Com NPM
npm run start

# Com Yarn
yarn start
```
